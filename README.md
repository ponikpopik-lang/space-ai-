 SPACE AI – Автономный спутник

Полнофункциональный автономный спутник для демонстрации принципов работы космических аппаратов и образовательных целей.

О проекте

Проект **SPACE AI** представляет собой кубсат-спутник с автономными системами, сенсорами и исполнительными механизмами.  
Цель проекта — показать работу автономных космических систем и использовать его в образовательных и исследовательских целях.  

Проект включает:
  Инженерную документацию и схемы.
  Презентацию проекта.
  Фотографию макета спутника.  

Документация включает:
  Технические характеристики спутника
  Описание сенсоров и исполнительных механизмов
  Систему управления и алгоритмы автономной работы
  Планы по дальнейшему развитию
Основные функции

  Автономное управление и мониторинг систем спутника
  Работа сенсоров для сбора данных
  Визуализация и хранение данных на борту
  Возможность демонстрации образовательных экспериментов
